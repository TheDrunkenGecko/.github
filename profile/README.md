## About Us

**The Drunken Gecko** is an independent software studio focused on building practical, high-quality software products and developer tooling.

Founded by **Dayton Jones**, our mission is simple:

> Build useful things. Solve real problems. Ship software we're proud of.

We specialize in creating software and services that help engineers, operators, and businesses work smarter through automation, cloud-native technologies, and AI.

---

## What We Build

### 🚀 SaaS Products

Applications designed to solve real-world problems with a focus on reliability, usability, and developer experience.

### 🤖 AI & LLM Applications

Intelligent assistants, workflow automation, and AI-enhanced products that augment human capabilities rather than replace them.

### ☁️ Cloud & Platform Engineering

Infrastructure, deployment platforms, and operational tooling built around modern cloud-native practices.

### ⚙️ Automation & DevOps Tooling

Tools that eliminate repetitive work, improve reliability, and help teams move faster.
